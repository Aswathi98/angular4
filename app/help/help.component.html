Help

Privileges
  • Admin: Given the privileges to upload,edit,send mail and register.The Admin is given sole access to the entire application.
  • HR: Given the privileges to edit and send mail
  • Recruiter: Given the privilges to send mail and edit the columns except litmus status,litmus batch,BGV status

Upload

While uploading excel sheet points to note:

  • Download the sample excelsheet from the webpage after downloading make sure date of drive cell is of type text
  • Click on browse button to select the filled excel sheet and then click on the upload button.The excel sheet will be uploaded.

View/Edit
  • The Admin can filter,sort and group the selected students into a batch using the Add to batch option.When you click on Add to Batch option a pop up is envoked where the user can provide a unique batch name. If you are entering an already existing batch make sure to spell the batch name correctly to avoid a new batch being created
  • The Recruiter can filter,sort the table .Editing is also possible except litmus batch,litmus status,BGV columns.The Add to batch option is disabled for Recruiter
  • The HR can filter,sort and group the selected students into a batch using the Add to batch option.Also has the privilge to edit all the columns.

SendMail
  • Click on the dropdown to select the required batch
  • Click on the template dropdown to select the type of mail to be send to that particular batch
  • To add a new Template click on add to template where the user is required to provide template type and template in html format.
